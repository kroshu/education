# education

[![Publish docs via GitHub Pages](https://github.com/kroshu/education/actions/workflows/github-pages.yaml/badge.svg)](https://github.com/kroshu/education/actions/workflows/github-pages.yaml) [![pages-build-deployment](https://github.com/kroshu/education/actions/workflows/pages/pages-build-deployment/badge.svg)](https://github.com/kroshu/education/actions/workflows/pages/pages-build-deployment)

This repository contains kroshu-related educational materials, including seminar task descriptions and related resources.

The project documentation website is generated as a static webpage using [MkDocs](https://www.mkdocs.org/).

## Local preview

Run the MkDocs development server from the repository root:

```shell
python -m mkdocs serve
```

Then open http://127.0.0.1:8000 in your browser.
